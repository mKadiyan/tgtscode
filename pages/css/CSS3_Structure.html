Css structure is made up of basically three things:
  • Selector
  • Property Name
  • Property Value
    •


    Selector

    A selector is something that identify location inside the html page where property needs to be set. h1tablebody.myclass#myId are the common example of selector.

    Property

    A property is something that tells what kind of style to be applied. displaytext-alignbackground-colormargin-bottommargin-top etc. are the common example of property.

    Value

    A value is something that defines a property.

    In the below example h1 is the selector, border and background-color are the properties and 1px red are the properties of border and background-color respectively.
    h1 { border: 1px; background-color: red; }
    Morever pair of property and value is called the declaration. For Instance, In above example border: 1px; is the declaration;
    In a single declaration , we can even define more the one value of a property. eg: border: 1px dotted red;
