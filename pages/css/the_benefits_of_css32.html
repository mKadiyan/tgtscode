Streamline Design


Earlier while designing the website, we do use the photoshop or other editor to add the look and feel of a website.This process takes more time if new features need to be added or when we try to chnage the design of a website. With css 3, We can directly make changes in the web browser and test our new design. Whenever a new requirement comes that required a change in design, we can directly modified the css and new design is ready. This is not only easy but also save lot of time required to testing and designing. It works like a magic.


Reduced Workarounds and Hacks


With the Introduction of css 3, we can do a lot of things with creating a workaround in javascript or html. css 3 comes with predefined property which we can use to style the webpage.

Some of them are mentioned below :

  • Rich Web fonts : Css provide us with many properties using which we can directly play with fonts in our webpage.
  • Reduced additional markup : With the help of css 3 , we can directly specify some of the property like display: table; display: table-cell; display: table-caption; etc.
  • Animation : Css is very rich in providing animation hence reduce heavy scripting needed for animation.
  • Gradients on the fly : We can use pattern and gradient on the fly using css 3.
  • Opacity and alpha channels: : Create transparency without images using opacity or rgba.
  • Corners and border styling: : Using different css property , we can style the border as round or straight eg. using border-radius .

Loads Page Faster

If you want to change design of the web page, Its a lot easier to change in the css then in html page.
