Css (Cascading style sheets) is basically used to style the html page. Using css one can greatly improve the look and feel of the web page without much effort.

Css was first introduce by Hakon Wium Lie on October 10, 1994 and W3C Recommendation on December 17, 1996. Here is the history of css specification.

CSS 1 : Published on December 17, 1996. Hakon Wium Lie and Bert Bos
  • Font properties such as typeface and emphasis
  • Color of text, backgrounds, and other elements
  • Text attributes such as spacing between words, letters, and lines of text
  • Alignment of text, images, tables and other elements
  • Margin, border, padding, and positioning for most elements
  • Unique identification and generic classification of groups of attributes
CSS 2 : Published on May, 1998.
  • absolute property added
  • fixed positioning property added
  • z-index property added
  • media types property added
  • support for aural style sheets
CSS 2.1 : Published on February 25, 2004.
  • fixed error of css 2.
CSS 3 : Published on February 25, 2004.
  • Introduce the concept of modularities.
  • css3-background
  • css3-box
  • css-cascade-3
  • css3-color
  • css3-content
  • css-fonts-3
  • css3-gcpm
  • css3-layout
  • css3-mediaqueries
  • css3-page
  • css3-selectors
  • css3-ui
CSS 4.
  • As of now there is no single integrated spcification for css4.
