For using css with html, we always have the option whether to embed the style within the html page or use external source. Below are the usage example :

Internal Stylesheet

In case of internal style sheet, we provide the css style code inside the html page only. Below is the example:

<html> <head> <title>lt;/title> <style type="text/css"> h1,h2 { text-align: center; } </style> </head> </html>

External Stylesheet

External stylesheets provide us a better way to keep the style and html concern separate.It's always good to keep these concern separatly.You can create the external css file using any text editor and save the file using .css extension.

External stylesheet for a html page can be achieve in two ways. Using style and link. Below are the examples:

mystyle.css
.notes { margin-top: 10px; margin-bottom: 10px; background-color: lightgreen; } .code { white-space: pre; background-color: antiquewhite; margin-bottom: 0px; margin-top: 0px; color: brown; } details p{ background-color: rgba(246, 252, 153, 0.69); }

index_style.html
<html> <head> <title>lt;/title> <style type="text/css">@import url('mystyle.css')</style> </head> </html>

index_link.html
<html> <head> <title>lt;/title> <style type="text/css" href="mystyle.css"></style> </head> </html>

Difference between css style and link

style import the external stylesheet and merge with the html page while link just provide the reference of the external style sheet and keeps the style and html code separate.

Inline Styles

Inline styles are those which we can directly embed with the html tag. see example below :

<p style="color: #ff0000;">Some Text</p>
